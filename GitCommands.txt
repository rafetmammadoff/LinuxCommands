Git Commands (Git komandalari)
git init (Her hansi folderi git ile elaqelendirmek)
git status (Deyishikliklerimizi gormek ucun)
git commit -m "message"  (Commit mesajimizi yaziriq)
git add . (Butun deyishiklikleri stage merhelesine salir)
git push origin + branchName (Githuba kodlari gondermek ucun)
git pull (githubdaki deyshiklikleri dartmaq ucun)
git log (commitleri gormek ucun)
git revert + commitId (commiti geriye qaytarir)
git reset --soft HEAD~ + (Sonnan nece commiti silmek isteyirsiz)
